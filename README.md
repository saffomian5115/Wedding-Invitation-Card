## ✨ About

This project includes **9 unique design variations** of the same invitation, each built with a different UI design philosophy (Neumorphism, Glassmorphism, Neubrutalism, Gothic, Material Design, and more), so you can pick the aesthetic that fits your vibe.

## 🚀 Features

- **Animated gate intro** — tap/click to dramatically open ornate doors revealing the invitation
- **Floating petals & golden particles** for ambient atmosphere
- **Background music toggle** with animated equalizer
- **Scratch-to-reveal card** for the wedding date
- **Auto-sliding photo gallery** with captions
- **Live countdown timer** to the big day
- **Event timeline** (Mehndi, Baraat & Nikkah, Walima, Reception)
- **Embedded Google Map** with directions button
- **RSVP / guest wishes wall** — visitors can leave messages live
- **Collaborative photo booth** — guests upload photos into a shared wedding frame
- **Smooth scroll-reveal animations** throughout
- **Fully responsive** for mobile and desktop
- **Urdu/Arabic calligraphy** accents (Noto Nastaliq Urdu, Tajawal fonts)

## 🛠️ Tech Stack

- Pure **HTML, CSS, and vanilla JavaScript** — no frameworks, no build step
- Google Fonts (Playfair Display, Cormorant Garamond, Space Grotesk, Tajawal, Noto Nastaliq Urdu, etc.)
- Canvas API for the scratch-card and photo booth compositing
- Intersection Observer for scroll animations

## 📂 Usage

1. Clone or download this repository
2. Open `index.html` in a browser to view the design gallery
3. Or open any individual `wedding invitation *.html` file directly
4. To customize for your own event, edit the names, dates, venue, and images directly in the HTML/CSS
